# 사용 메뉴얼

## 1. 제일 쉬운 실행 방법

### 방법 A) 바로 실행 (권장)
`start_threadpilot_ui.bat` 더블클릭

자동으로 처리되는 항목:
- `dist/threadpilot.exe`가 있으면 EXE로 바로 UI 실행
- EXE가 없으면 Python 모드로 전환
- `.venv` 생성
- 기본 의존성 설치 (`requirements.txt`)
- UI 실행

### 방법 B) 기본 설치를 전부 한 번에
`setup_default.bat` 더블클릭

자동으로 처리되는 항목:
- `.venv` 생성
- `pip` 업그레이드
- 앱 의존성 설치
- `pyinstaller` 설치
- EXE 빌드 (`dist/threadpilot.exe`)
- UI 실행

## 2. UI 사용 순서 (초보자 기준)
1. Health Check 버튼 클릭
2. 요구사항 파일 경로 입력 후 Bootstrap 실행
3. PR ID 입력 후 Review 실행
4. 아래 JSON 결과를 복사해 보고/공유

## 3. 수동 실행(원할 때)
```bash
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
python -m threadpilot.main ui
```

## 4. 주요 명령어
```bash
python -m threadpilot.main bootstrap --requirements-file 요구사항.md
python -m threadpilot.main review --pr 123
python -m threadpilot.main review --pr 123 --live
```

## 5. API 엔드포인트
- `GET /api/health`
- `POST /api/bootstrap`
- `POST /api/review/{pr_id}?live=false`

## 6. 데이터 저장 위치
- `threadpilot/data/project_state.json`
- `threadpilot/data/requirements.json`
- `threadpilot/data/todo_list.md`
- `threadpilot/data/chroma/`
